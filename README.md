# Python Zip Cracker

#### Created by: Nurfian Qodar

## Installation

-   ### Create virtual environment

```sh
python3 -m venv venv
source venv/bin/activate
```

-   ### Install requirements

```sh
pip install -r requirements.txt
```

-   ### Enjoy

```sh
python3 bfzip -i path/to/zip -l path/to/wordlist
```
